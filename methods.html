Methods

First, we took a video of each side of Queen Street for reference. Throughout the process some stores closed,like Toys R. Us and Halifax. We built a vast dataset on Google Sheets so we could all access and edit it, using journalistic sources and company reports. We explored ways to interact with the Google Maps Java Script API to create clickable overlays, and then code in the information we wanted from our data set to populate the pop ups. This answered part of our question, "who owns Queen Street?" We also used the land registry database from the UK government to discern who owns the land and property, and found that many English public offices hold that land. We conducted community interviews to understand how those owners interact with the Queen Street business community. Much of our dataset comes from the company reports from these businesses, which can be found compiled here
